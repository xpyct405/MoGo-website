BrainsCloud Gulp!

1. Создаём папку проекта.
2. Закидываем туда src, gulpfile.js, package.json
3. Запускаем VScode и терминале пишем:

npm i для установки нод модулей. Вначале каждого проекта.
gulp для обычной работы, верстки
gulp build в самом конце когда надо выводить проект


1. src/layouts/default.html Формируется скелет сайта (Только один файл в папке)
2. src/partials/ тут куски кода, которые формируют основную! страницу, например aside, footer, header, всякие блоки в середине! А вставляются в default.html так
      
        <!-- Header -->
        {{> header }}

        <!-- Sidebar -->
        {{> sidebar }}

3. src html файлы те которые- идут вместо основной страницы (вместо той части которую формирует src/parials body модуль!), например те на которые введут ссылки, типо окно регистрации, отдельное окно для связи, Во всех этих окнах останется весь код (шапка, футер, эсайд) кроме   {{> body }}

4.Шрифты
Шрифты заранее подключаются через google fonts и пишем в шапку head.html


5. Как указывать ссылки на картинки:
html =>  <img src="assets/img/NAME.png" alt="NAME png">
css =>   background: url('../../assets/img/NAME.png') center no-repeat; 

6. Картинки разбиваем по папкам!


7. Подключение FontAwesome- икноки от бутстрапа перед ИКОНКАМИ ДОЛЖНО БЫТЬ  fas fa fab если отображаются квадратики ИЛИ 
body i { font-family:'FontAwesome'!important;}
